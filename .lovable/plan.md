
# Faisal Aftab-Inspired Portfolio Site Clone

A single-page personal portfolio website for your client, closely matching the design and layout of faisalaftab.com. All content will use placeholders for now.

## Design & Theme
- **Dark background** (dark charcoal/near-black) with **teal/cyan accent colors** for headings and highlights
- Bold, uppercase typography for section headers
- Clean, professional feel targeting investors/executives

## Sections (top to bottom)

### 1. Hero Section
- Large bold name in teal/cyan gradient text
- Subtitle with professional title and tagline
- Social media icon links (Facebook, LinkedIn, Instagram, Twitter)
- "Learn More" outlined button
- Portrait photo on the right side with a large watermark-style "VC" text behind it
- Scrolling ticker/marquee bar at the bottom with key titles

### 2. Introduction Section
- "INTRODUCTION" heading on the left
- Multi-paragraph bio text on the right
- Portrait image alongside the text
- "Let's Know More" call-to-action button

### 3. Portfolio/Investments Section
- Grid of company logos (placeholder logo images)
- Organized in multiple rows

### 4. Press, Talks & Conferences Section
- Grid of thumbnail cards linking to external appearances
- Each card has an image thumbnail and a bold title
- Scrollable or multi-row layout

### 5. Board Seats Section
- List of board positions with company name, role, and date range
- Clean card or list layout with links to company websites

### 6. Testimonials Section
- "What Founders & Peers Say" heading
- Carousel/slider of testimonials with photo, quote, name, and title

### 7. Footer
- Navigation links (About, Investments, Board Seats, Feedback)
- Social media icons
- Clean, minimal dark footer

## Technical Approach
- Fully static, no backend needed
- Single-page React app with smooth scroll navigation
- Responsive design (desktop + mobile)
- Subtle animations (marquee ticker, testimonial carousel)
- Placeholder content throughout, ready for client to swap in real info
